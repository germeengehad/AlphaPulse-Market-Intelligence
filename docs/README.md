# AlphaPulse

AlphaPulse is a real-world financial prediction system that integrates
market data, NLP-based news sentiment, machine learning models, and a 
dynamic trading allocation engine designed to predict S&P 500 excess 
returns and generate leverage signals between 0 and 2.

This system is based on real hedge-fund workflows:
- Market data ingestion (Yahoo Finance)
- SQL storage (PostgreSQL)
- Feature engineering
- NLP sentiment from news
- ML forecasting
- Dynamic exposure allocation
- Risk management
- Backtesting
- Live daily updates
